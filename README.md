## k8s
#### 로컬 환경 Kubernetes 구조 예제
- /k8s/my-app
  - 단순한 Kubernetes ClusterIP Service(Node.js)
- /k8s/nginx/config
  - Kubernetes NodePort와 Nginx 설정을 사용하여 my-app에 연결시킨 예제
- /k8s/nginx/ingress
  - Kubernetes Ingress Controller와 Ingress를 사용하여 my-app에 연결시킨 예제
